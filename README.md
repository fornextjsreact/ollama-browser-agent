# Multi-Agent Browser Control System

This repository provides a multi-agent browser control system using local Ollama. The purpose of this system is to enable multiple agents to automate browser tasks in a collaborative manner.

## Features
- Control multiple browsers simultaneously
- Integrate with Ollama local server for agent communication
- Customizable agent behavior
- Easy setup and deployment

## Getting Started
To get started with the multi-agent browser control system, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/fornextjsreact/ollama-browser-agent.git
   ```
2. Navigate to the project directory:
   ```sh
   cd ollama-browser-agent
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the server:
   ```sh
   npm start
   ```

## Usage
Once the server is running, you can begin adding agents to control your browser tasks. Refer to the documentation for detailed instructions on how to configure each agent and their respective actions.
